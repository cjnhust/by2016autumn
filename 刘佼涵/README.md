                                     总结  
    回想这次天气预报app的开发，收获还是很多的。从之前对app的开发一无所知，到现在有了一些了解。刚拿到这个实习任务的时候，真的是一脸懵逼，完全不知道要从哪里开始下手。花了好久把Java环境弄好，一开始不知道，使用eclipse，ADT下了好几个版本都是损坏的。最后才知道要用Android Studio和SDK。然后是是开发语言，虽然有面向过程语言的基础，但刚开始看面向对象好像是理解了，其实并没有。Java最基础的类和对象，没有通过实际操作，光是看看书本的概念是不能理解的。大概花了2天的时间，零零散散的看了书，程序也就能看个大概了。
    准备工作做了一些就是要开始开发了，刚开始不知道要从页面还是从网络数据请求开始，所以就随机开始从网络请求开始，刚开始对Android Studio并不熟悉，一个文件夹的位置，都要上百度看看在哪里。经过一系列努力，也只完成到SDK的初始化。
    于是我转向了界面的开发，刚开始只知道Button，TextView,Imagine View这些简单的控件，竟天真地以为所有页面都是这样一个个拼起来的，还很不好拼，要把他们拼整齐就花了好长时间。直到后来学长告诉我还有LinearLayout、RelativeLayout这样的布局，以及listview，然后我就开始了这些的学习。学习了他们简单的应用和Listview引入布局文件。然后我想弄好的这些Activity如何建立联系呢？后来得知有intent方法。于是学习了使用显示Intent。以及给listview的Item加入点击时间。最后我学习了如何使用通知，从书本上看来的方法中setLatestEventInfo已经不能使用了，我找到了Builder类进行替代，实现了通知的使用。
    最后，我回到了最初的问题如何使用网络请求，数据的解析和请求，看得有点乱，百度上方法各不相同，知道截至时间还是没有完成，很遗憾。
    这次开发经历，有遗憾，但收获更多。对当初一无所知的app开发有了了解。
